---
rule_id: 1009
rule_category: naming-conventions
title: Namen einer Klasse oder Enumeration nicht in den Membern wiederholen
severity: 2
---
```
class Employee
{
    // Falsch!
    static GetEmployee() {...}
    DeleteEmployee() {...}

    // Richtig
    static Get() {...}
    Remove() {...}

    // Auch richtig
    AddNewJob() {...}
    RegisterForMeeting() {...}
}
```
